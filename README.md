# Hi-Fi Store Web App

## Tech Stack
- Django
- PostgreSQL
- Stripe
- django-allauth
- Whitenoise

## Setup
1. Clone repo
2. Create virtual environment
3. Install dependencies from requirements.txt
4. Run `python manage.py runserver`

## Purpose
A fictional e-commerce site for hi-fi equipment, built as a final project.
